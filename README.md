# Europa Park Wartezeiten
Empfängt die aktuellen Wartezeiten von Attraktionen im Europa Park Rust. Durch Geofencing wird diese Funktion durch den Park eigentlich gesperrt.

## Ausführen
- Auf den Server laden und starten!
